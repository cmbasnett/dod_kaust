# Build Settings
The map is built with [Zoner's Half-Life Tools](http://zhlt.info/), and uses the following settings.


    hlcsg.exe
    hlbsp.exe -nohull2
    hlvis.exe
    hlrad.exe -bounce 16 -smooth 200 -smooth2 200 -texdata 16000 -blur 1.5 -minlight 8 -gamma 0.5
